# Frontend

## Visual Style
- Aesthetic: clean, minimal, dark theme — premium SaaS feel (Linear/Vercel)
- Typography: Inter, tight letter-spacing for headings
- Colors: oklch scale, dark backgrounds, accent colors per category
- Border-radius: 12px cards, smooth and consistent

## Layout
- Hebrew RTL interface
- Responsive — mobile-first
- CSS Grid for page layout, Flexbox for component internals

## Accessibility (baseline)
- Semantic HTML elements
- All interactive elements keyboard-focusable
- aria-labels on icon-only buttons
