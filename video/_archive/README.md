# _archive - קבצים ישנים

קבצים אלו הוחלפו והם נשמרים רק לצורכי היסטוריה. **אין להשתמש בהם.**

## התסריט הרשמי
- מקור יחיד לאמת: `src/config/scenes.ts`
- קבצי ה-MP3 המופקים: `public/assets/audio/narration/`

## מה יש כאן
- `generate-ssml-narration.py` - סקריפט ישן שהפיק SSML עם טון היסטרי ("תקשיבו, גיליתי משהו מטורף!"). הוחלף בתסריט רגוע ואינפורמטיבי ב-scenes.ts.
- `ssml/` - קבצי ה-SSML שנוצרו מהסקריפט לעיל. לא בשימוש.
- `gemini-script-v3.txt`, `gemini-script-v3-utf8.txt` - טיוטות תסריט ישנות מ-Gemini. הוחלפו.

## הערה
קבצי ה-MP3 הקיימים ב-`public/assets/audio/narration/` תואמים את הנוסח הרשמי ב-`scenes.ts` ולא את הקבצים כאן.
